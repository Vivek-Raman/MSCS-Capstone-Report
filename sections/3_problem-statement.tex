\section{Problem Statement}

\LaTeX\ provides unparalleled quality and precision, but is very difficult to learn.
Both new writers and those who are already familiar with \LaTeX\ may spend large amounts of time doing repetitious work.
In contrast, many popular writing software tools such as Microsoft Word and Google Docs provide ease of use; however, these tools do not allow enough precision and detail to meet the needs of high-quality academic publishing.
Thus, the inefficiencies present due to this discrepancy impact the entire publishing process from author to publisher.
Authors can experience days of back-and-forth communication with publishers regarding minor errors that would have never occurred if there were better tools available.
Therefore, researchers are forced to choose between spending weeks or months learning how to properly utilize \LaTeX\ so that they can produce high-quality documents with maximum flexibility or settling for lower-quality documents.

Spartan Write fills the gap between quality, functionality, and ease of use by utilizing Artificial Intelligence to automate the technical aspects of formatting.
Spartan Write allows both novice users and graduate level students to create documents that are ready for publication with minimal effort.

The following is a list of functional requirements for Spartan Write.

\begin{itemize}
\item Allow creation of \LaTeX\ projects based upon pre-existing templates.
\item Provide access to open templates created by publishers.
\item Present an interface for interaction with an artificial intelligence (AI) agent which can manage the source code of the document.
\item Provide means to directly modify the source code of the document by the user.
\item Support uploading images to the AI agent so that they can be attached to the document or so that the AI agent can convert them into LaTeX code.
\item Evaluate and determine which LLM is optimal for each job via a reproducible testing suite.
\item Create a user manual designed for graduate-level students to enable rapid onboarding and productivity.
\end{itemize}
