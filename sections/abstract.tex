\begin{abstract}
LaTeX gives researchers precise control over academic documents, but it can be hard to learn and slow to use.
Spartan Write is a desktop writing tool that lets users edit LaTeX projects through natural language instead of writing every command by hand.
Users can draft text, create tables, add figures, and fix formatting issues from a chat interface while still keeping access to the source and compiled preview.
The system combines a Tauri/React frontend, a local FastAPI sidecar for files and compilation, and a LangGraph agent for document edits.

This report presents the architecture and evaluates Spartan Write with a reproducible benchmark across multiple LLM endpoints.
The benchmark measures task completion, runtime, token use, cost, and tool behavior.
Across 119 test cases, the system completed 98.3\% of tasks.
The results show large differences in cost and speed between providers, mostly due to how each model uses tools.
Gemini 3 Flash is the best default in this benchmark because it completed every task, had the fastest average runtime at 12.9s, and kept mean cost low at \$0.0094 per run.
\end{abstract}
